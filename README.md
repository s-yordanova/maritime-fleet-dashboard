# Fleet Manning Dashboard

Interactive maritime dashboard built with React, TypeScript and Vite.

## Features

- Interactive world map
- Vessel tracking and route visualization
- Manning agent locations
- Fleet activity ticker
- Operational summary cards
- Animated dashboard experience
- Responsive layout

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand
- MapLibre GL

## Getting Started

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Overview

The dashboard provides a global overview of vessels and manning agents through an interactive map interface. Users can inspect vessel details, view manning agent information, monitor fleet activity, and visualize vessel routes worldwide.
